## Installation

###  Manual Install

1. Install dependencies:

    - git
    - make
    - librsvg2-bin

2. Run the following commands as normal user:

    ```
    git clone https://github.com/PapirusDevelopmentTeam/papirus-calibre-theme.git
    cd papirus-calibre-theme
    make build
    make install
    ```

3. Start Calibre and enjoy.
